# Decision Log

Append-only. Never modify existing entries. Each decision gets a unique D-* ID.

---

## Decision Template

```markdown
## D-NNN — [Short title]
**Date:** YYYY-MM-DD
**Status:** accepted | superseded-by D-XXX
**Context:** Why did this decision need to be made?
**Decision:** What was decided?
**Alternatives considered:** What else was an option and why rejected?
**Consequences:** What follows from this decision?
**Requirement refs:** R-001, R-002
```
